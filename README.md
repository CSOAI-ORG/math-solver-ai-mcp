# math-solver-ai-mcp

MCP server for math and statistics tools.

## Tools

- **solve_equation** — Solve linear and quadratic equations
- **statistics_summary** — Comprehensive statistical analysis
- **matrix_operations** — Matrix math (multiply, add, transpose, determinant)
- **probability_calculator** — Probability calculations (binomial, combinations, permutations)

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
